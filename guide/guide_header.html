헤더 가이드 (Headers)

유형 설명 표기
메인 헤더 • 메인 헤더는 서비스 명, 알림 버튼, 전체 메뉴 버튼으로 구성

소속명 출입증
서브 헤더 Type A • 이전 버튼, 페이지 타이틀(가운데 정렬), 홈 버튼, 전체 메뉴 버튼으로 구성

소속명 출입증
서브 헤더 Type B • 이전 버튼, 페이지 타이틀(가운데 정렬)로 구성

소속명 출입증
서브 헤더 Type C • 버튼 및 타이틀이 없는 경우
• 프로세스 완료 화면에서 사용
서브 헤더 Type D • 이전 버튼, 홈 버튼, 전체 메뉴 버튼으로 구성
• 헤더영역까지 화면 스크롤 시 화면에 있는 타이틀 페이지타이틀로 변경 (서브 헤더 Type A)
전체메뉴 헤더 • 홈 버튼, 닫기 버튼으로 구성
닫기버튼 헤더 • 닫기 버튼으로 구성
Full Popup 헤더 • 닫기 버튼으로 구성
Bottom Sheet Popup 헤더 • 닫기 버튼과 팝업타이틀로 구성
• 혹은 버튼 및 타이틀이 없는 경우

소속명 출입증
Popup 헤더 • 닫기 버튼 구성
• 혹은 버튼이 없는 경우
